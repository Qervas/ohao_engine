# OHAO Engine

OHAO Engine is a modern, high-performance game engine developed for Linux platforms, focusing on advanced rendering techniques, procedural generation, and physics simulation.


## Building

### Prerequisites
- CMake 3.20+
- Vulkan SDK
- GLFW3

### Build Steps
```bash
mkdir build
cd build
cmake ..
make
```

### Running
```bash
./ohao_engine
```
